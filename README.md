# Flight BGA-044 (React + Vite + Tailwind)

## Run locally

1. Install Node.js (v18+ recommended)
2. In this folder:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Push to GitHub

```bash
git init
git add .
git commit -m "Initial commit"
# create a repo on GitHub, then:
git remote add origin <YOUR_REPO_URL>
git push -u origin main
```

## (Optional) Deploy to GitHub Pages

Easiest: deploy with Netlify/Vercel.
If you want GitHub Pages, you can use a GitHub Action to build and publish `dist/`.
